# Sistema de Contexto - HandsomeTranscribe

Este directorio centraliza contexto tecnico util para desarrollo asistido por IA y trabajo del equipo.

## Capas

- `project/`: informacion estable del proyecto (arquitectura, patrones, convenciones, restricciones).
- `active/`: contexto de trabajo por dominio/feature en curso.
- `sessions/`: notas temporales de implementacion (efimeras).
- `archive/`: historico de decisiones o investigaciones cerradas.

## Inicio rapido

Leer primero:

1. `.context/project/architecture.md`
2. `.context/project/patterns.md`
3. `.context/project/conventions.md`
4. `.context/project/constraints.md`

## Regla de mantenimiento

Si un archivo fue migrado de otro proyecto y no aplica, actualizarlo o moverlo a `archive/`.
